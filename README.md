# cn-ex · 中国制霸

中国省级行政区域「制霸」标记工具：点亮去过的省，分数实时计算，**用 URL 参数 `?l=` 分享状态**。

体验与交互主要参考 [JapanEx 制县等级](https://zhung.com.tw/japanex/)：简化方块地图、等级色块、点选上色、可分享的状态串。  
实现为轻量静态页（无后端、无账号）。

## 打开

```bash
cd cn-ex
pnpm dev
# 或：python3 -m http.server 5178
```

浏览器访问：<http://127.0.0.1:5178/>

带状态示例：

```text
http://127.0.0.1:5178/?l=0050000000000033524303013330454033
```

可选标题：`?l=...&t=我的足迹`（会显示在地图标题）。

## 功能

| 能力 | 说明 |
| --- | --- |
| 点省设等级 | 居住 5 / 短居 4 / 游玩 3 / 落脚 2 / 路过 1 / 没去过 0 |
| `?l=` 分享 | 34 位数字串，每位一省；改动时 `history.replaceState` 同步地址栏 |
| localStorage | key：`cn-ex-levels`；无 URL 时兜底恢复 |
| 复制链接 | 一键复制当前完整 URL |
| 保存图片 | SVG → Canvas → PNG 预览/下载 |
| 清空 | 重置全部为 0 |

### 状态优先级

1. 打开时若有 `?l=` → 以 URL 为准，并写回 localStorage  
2. 否则读 localStorage  
3. 每次改等级 → 更新 localStorage + 地址栏 `?l=`

### 省份顺序（勿改）

顺序一旦发布就应冻结，否则旧链接会错位：

```text
黑龙江 甘肃 吉林 内蒙古 山东 河北 北京 天津
西藏 新疆 河南 安徽 山西 湖北 青海 辽宁
广东 江苏 江西 浙江 福建 上海 陕西 湖南
广西 香港 澳门 贵州 重庆 四川 云南 宁夏
台湾 海南
```

## 等级约定

### 色阶（足迹深度）

有序色：冷 → 暖，对应「路过的疏离」到「居住的扎根」，避免红绿灯语义。

| 等级 | 色值 | 气质 |
| --- | --- | --- |
| 0 没去过 | `#F3F5F7` | 纸白灰，未标记 |
| 1 路过 | `#8FA6C4` | 钢蓝，短暂掠过 |
| 2 落脚 | `#5E9E98` | 雾青，短暂落脚 |
| 3 游玩 | `#E2B24A` | 琥珀，休闲记忆 |
| 4 短居 | `#D96B42` | 陶橙，临时归属 |
| 5 居住 | `#C0394A` | 胭脂，扎根 |
| 地图底 | `#B4CDEA` | 柔雾蓝，托色不抢戏 |



与 JapanEx 语义对齐的个人标准示例：

日常标签 + JapanEx 式备注：

- **5 居住（居住过）**
- **4 短居（住宿过）**
- **3 游玩（游玩过）**
- **2 落脚（休息、换车等）**
- **1 路过（路过）**
- **0 没去过**

## 地图

采用 **简化方块省图**（制霸玩具图，不是测绘地图）：

- 保留省区相对方位与相邻关系，形状故意方正，方便点选和分享  
- 34 省可标记；`?l=` 顺序固定  
- 手改 SVG：编辑 `src/map-block.svg` 后执行 `python3 scripts/build-map.py`
